* What is DOM ?

When a web page is loaded, the browser creates a Document Object Model of the page.

We can find it under inspect by right click and then Network, DOM content loaded with details.

With the object model, JavaScript gets all the power it needs to create dynamic HTML

* JavaScript can change all the HTML elements in the page
* JavaScript can change all the HTML attributes in the page
* JavaScript can change all the CSS styles in the page
* JavaScript can remove existing HTML elements and attributes
* JavaScript can add new HTML elements and attributes
* JavaScript can react to all existing HTML events in the page
* JavaScript can create new HTML events in the page

Types of DOM

* Core DOM - standard model for all document types
* XML DOM - standard model for XML documents
* HTML DOM - standard model for HTML documents

* Comparison between let, const and var

Features of let:

a.) Variables defined with let cannot be Redeclared, with var we can.

let x = "John Doe";
let x = 0;
// SyntaxError: 'x' has already been declared

b.) Variables defined with let must be Declared before use.

c.) Variables defined with let have Block Scope.

Variables declared inside a { } block cannot be accessed from outside the block:

{ let x = 2; }
// x can NOT be used here

d.) Redeclaring a variable inside a block will not redeclare the variable outside the block:

let x = 10;
// Here x is 10
{
let x = 2;
// Here x is 2
}
// Here x is 10

Features of const:

a.) Variables defined with const cannot be Redeclared just like let with var we can.

const x = "John Doe";
const x = 0;
// SyntaxError: 'x' has already been declared

b.) A const variable cannot be reassigned:

const PI = 3.141592653589793; PI = 3.14; // This will give an error PI = PI + 10; // This will also give an error

* HTML EVENT METHODS

a.) addEventListner() Method

The addEventListener() method attaches an event handler to the specified element.We can add many event handlers to one element.

Syntax: element.addEventListener(event, function, useCapture);

The first parameter is the type of the event (like "click" or "mousedown" or any other HTML DOM Event.)

The second parameter is the function we want to call when the event occurs.

The third parameter is a boolean value specifying whether to use event capturing. This parameter is optional.
