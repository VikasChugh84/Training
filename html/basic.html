HTML is the hyper text markup language for creating Web pages, consists of series of elements that tell the browser to display the content.

a.) Elements

The HTML element is everything from the start tag to the end tag.

For example: (Content goes here...).

b.) Doctype

It declaration represents the document type, and helps browsers to display web pages correctly.It must only appear once, at the top of the page (before any HTML tags).

c.) Headings

These are are defined with the (h1 to h6) tags, h1 defines the most important heading and h6 defines the least important heading.

d.) Paragraphs

These are are defined with the (p tag), anything paragraph is defined is under (p tag).

The browser will automatically remove any extra spaces and lines when the page is displayed.

e.) Links - Details

These are are defined with (a tag).

Link: www.berylsystems.com

f.) Images

These are are defined with (img tag)

Image Attributes: The source file (src) is image location, alternative text (alt) is optional, width and height depends we defined.
