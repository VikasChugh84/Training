a.) head Element

The HTML (head) element is a container for the following elements: title, style, meta, link, script and base.

b.) title

The title element defines a title in the browser toolbar -- provides a title for the page when it is added to favorites -- displays a title for the page in search engine-results.

c.) link

The link element defines the relationship between the current document and an external resource.The link tag is most often used to link to external stylesheets.

d.) meta

The metadata will not be displayed on the page, but are used by browsers (how to display cosntent or reload page), by search engines (keywords), and other web services.

* Define the character set used.

* Define keywords for search engines.

* Define a description of your web page.

* Define the author of a page.

* Refresh document every 30 seconds.

* Setting the viewport to make your website look good on all devices.

e.) Setting Viewport

The viewport is the user's visible area of a web page. It varies with the device - it will be smaller on a mobile phone than on a computer screen.

meta element should be included in all your web pages.

The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device).

The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.

e.) Script

The script element is used to define client-side JavaScripts.

e.) base

The base element specifies the base URL and/or target for all relative URLs in a page.The base tag must have either an href or a target attribute present, or both and there should be single base class element in a document.
