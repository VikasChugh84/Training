Attributes of Page Layout

Page layout is the part of graphic design that deals with the arrangement of visual elements on a page

a.) Header

The part of a front end which is used at the top of the page, (header tag) is used to add header section in web pages.

b.) Navigation Bar

The navigation bar is same as menu list. It is used to display the content information using hyperlink.

c.) Index/Sidebar(optional)

It holds additional information or advertisements and is not always necessary to be added into the page.

d.) Content Section

The content section is the main part where content is displayed.

e.) Footer

The footer section contains the contact information and other query related to web pages. The footer section always put on the bottom of the web pages. The (footer tag) is used to set the footer in web pages.
