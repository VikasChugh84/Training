a.) Introduction

Tables are used to arrange data into rows and columns.

* th - used to insert table header.

* tr - used to insert table row.

* td - used to insert table data.

* thead - It groups the header content in table.

* tbody - It groups the body content in table.

* tfoot - It groups the footer content in table.

* caption - used to display table name.

* colgroup - Specifies a group of one or more columns in a table for formatting.

* col - Specifies column properties for each column within a element.
